# Vue ToDo List

## Clone Project
```
git clone https://github.com/cokacolaa/todolist.git
```

## About
Project of a CRUD ToDo list made in vue.js

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
